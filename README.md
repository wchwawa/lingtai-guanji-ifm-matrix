## 当前版本已更新什么

当前页面已不是最早的单页 IFM Matrix worksheet，而是「灵台·观己」健康追踪 App 原型，已包含：

1. **观己首页**：以「灵台·观己」为主标题，定位为关注本心、共同成长的健康档案与矩阵报告工具。
2. **长期档案入口**：支持基础资料、生活方式、检查指标、批量文字档案导入与本地保存，用于形成连续健康资料。
3. **功能医学时间轴 / 矩阵模板**：保留官方 IFM Matrix 的时间轴与矩阵图结构，用于资料整理与专业沟通。
4. **观己故事报告**：报告不只填格子，而是把资料整理成「功能医学时间轴」「功能医学工具-矩阵」「观己故事」等段落。
5. **AI 辅助边界**：AI 只作资料整理、追问清单、沟通稿与风险提醒辅助；不替代医生、营养师或相关专业人员。
6. **导出与预览**：支持浏览器打印 / PDF 导出，适配桌面与手机查看。

## 使用方法

1. 打开在线预览地址。
2. 在页面中填写基础信息、生活方式、检查指标或批量文字档案。
3. 使用矩阵与时间轴整理前因、触发、介质、功能节点与生活方式线索。
4. 生成报告，重点阅读「观己故事」与追问清单。
5. 下一次回来继续追加资料，让档案逐步形成自己的身体年轮。
