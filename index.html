灵台·观己

关注本心 · 共同成长

把身体的故事，从根到果完整记录下来——长期健康数据档案 + 功能医学问卷与矩阵 + 内置离线功能医学分析报告。本工具为观察记录辅助，非诊断、非治疗、非处方。

灵台·观己 是健康记录与功能医学结构梳理的辅助工具。
所有内容为结构化观察整理，默认离线规则引擎；非诊断、非治疗、非用药 / 营养治疗处方；请以营养师 / 专业人员判断为准。

功能医学问卷 · IFM

健康自填问卷

凭直觉填写，几分钟即可完成。本问卷帮助把您的故事映射到功能医学结构，仅作观察梳理，非诊断。
长期健康数据档案 · RECORD

健康数据记录

长期、结构化地记录体格、饮食与各项生化指标。指标分组与参考范围仅作记录格位，不做医学解释或诊断；填写后可保存到本地档案，供营养师 / 专业人员查看。

批量文字档案导入 文本档案

可粘贴客户聊天记录、三日饮食打卡、既往体检摘要、营养咨询草稿等文字资料。此处只作本地档案整理，不自动诊断、不替代营养师判断。

⚕ 本模块仅为健康数据的结构化记录与趋势入口，不构成任何诊断、治疗或处方建议。各项参考区间因实验室与个体而异，具体解读请以检测机构报告及营养师 / 医生判断为准。
趋势与档案 · TREND

趋势与历史

查看已保存的数据档案与简单趋势入口，便于长期对照。趋势仅为数值变化的可视化整理，不做医学判断。
功能医学矩阵 · IFM MATRIX

我的功能医学矩阵

按 IFM 七大生理节点 + 中心情志（MES），把问卷勾选的感受归位。仅作整理梳理，不是体检分数、不是诊断。
功能医学分析报告 · SKILL ENGINE

功能医学分析报告

默认离线规则引擎生成。基于已选节点 / 已填数据做基本映射，输出数据摘要、IFM 矩阵、重点节点、资料缺口、追问清单、保守生活方式方向与红线提醒。供营养师 / 专业人员参考，不诊断、不治疗、不处方。
点击上方「生成功能医学分析报告」，由内置功能医学 skill / 规则引擎在本地生成报告。
引擎不依赖 LLM，不联网，不上传任何数据。

建议先完成「功能医学问卷」并在「健康数据记录」填入可用指标，报告会基于已填内容做映射。
AI 增强 · 可选 OPTIONAL

AI 增强（可选）

默认离线规则引擎；AI 仅作可选润色 / 摘要 / 追问生成，不参与红线判定、实验室异常判断、诊断或处方。开启需自备 Key、脱敏后发送，且任何免费额度都不稳定。
